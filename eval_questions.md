# RAG 评估问题样例

用于后续构建自动化评估集，可从“答案是否基于资料、关键字段是否命中、是否给出正确流程”三个维度评分。

| 问题 | 期望命中文档 | 关键点 |
| --- | --- | --- |
| 出差住宿一线城市最多报销多少钱？ | 报销制度 | 一线城市不超过600元/晚 |
| 单次网约车超过多少钱需要说明原因？ | 报销制度 | 超过200元需说明路线和业务原因 |
| 病假超过1天需要上传什么材料？ | 请假与考勤制度 | 诊断证明、病假建议书或挂号记录 |
| 调休应在加班后多久内使用？ | 请假与考勤制度 | 3个月内使用 |
| 采购金额超过5000元需要补充什么材料？ | 采购审批流程 | 至少两家供应商报价 |
| 超过20000元的采购审批链路是什么？ | 采购审批流程 | 采购委员会、财务负责人、总经理审批 |
| 生产数据库凭证属于什么级别的数据？ | 信息安全规范 | 敏感或机密数据 |
| 发现账号异常登录后多久内上报？ | 信息安全规范 | 1小时内 |
| 新员工试用期目标什么时候确认？ | 入职与转正流程 | 入职后7个工作日内 |
| P1工单多久内响应？ | 客户工单处理SOP | 30分钟内响应 |
